## Tech Stack

* Laravel
* SQLLite
* PHP

## Download Instruction

1. Clone the project.

```
git clone https://github.com/infinitypaul/sdui.git projectname
```


2. Install dependencies via composer.

```
composer install 
```

3. Migrate and seed the Database.

```
php artisan migrate --seed
```

4. Run Test.

```
php artisan test
```

5. Run php server.

```
php artisan serve
```

6. Available Command.

```
php artisan clean:news 
```


Enjoy!!
